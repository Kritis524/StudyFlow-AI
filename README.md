# 🚀 StudyFlow-AI
An AI-driven study planner designed to automate academic task management and optimize student productivity.

## 🛠️ Tech Stack
- **Frontend:** React + Vite, Tailwind CSS
- **Backend:** Python (FastAPI/Flask)
- **AI Integration:** Google Gemini API (RAG-based scheduling)
- **State Management:** React Hooks
- **Icons:** Lucide React / SVG

## ✨ Features
- **AI Planner:** Automatically generates study schedules based on syllabus parsing.
- **Adaptive Scheduling:** Predicts task difficulty and adjusts time blocks accordingly.
- **Focus Timer:** Built-in Pomodoro timer to maintain productivity.
- **Subject Tracking:** Organize and manage individual course modules.
- **Interactive Dashboard:** A clean, modern UI for a bird's-eye view of your academic goals.
